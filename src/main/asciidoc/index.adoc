*Current Active Endpoint*: https://bible-verse-api.herokuapp.com

This API is built on the http://jsonapi.org[JSON API] standard. The only function
available is `GET` with the various filters oulined below.

[[Formats]]
= Formats

Available formats:

* niv (default)
* cev
* nkjv
* amp
* beng
* kjv
* nlt
* ylt
* asv
* msg
* darby
* nasb
* nrsv
* esv
